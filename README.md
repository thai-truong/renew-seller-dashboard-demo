# README

## Configuration steps:
0. Set up `asdf` (or your preferred tool version manager) with:
  - Node.js (22.2.0 or above)
  - Ruby (3.2.3 or above)
  - Yarn (4.2.2 or above)
1. `bundle install`
2. `yarn install`

## To start up the app on localhost:3000
- Run `spring stop;rails s` in your terminal.
